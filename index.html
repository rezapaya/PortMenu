Portmenu.

A little script to help using Portage in Gentoo Linux. Please note, that I did not use Gentoo in many years, so this script may be not working anymore. But feel free to fork it!

Save it in /usr/bin/portmenu and make it executable. Then you're fine to run it!

$ mv portmenu.sh /usr/bin/portmenu
$ chmod +x /usr/bin/portmenu
$ portmenu
